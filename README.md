# CS-335 AI Agents for Bioinformatics

## Team Members
- Ozi Jawad
- Carter Struck
- Mack Hicks
- Sona James

---

Dealing with biological data (e.g., DNA sequences) often involves using specialized libraries for obtaining, processing, and interpreting the data, and typically requires assistance from trained bioinformaticians.  

The goal of this project is to explore creating an **AI Agent** that enables non-bioinformatics users to work easily with genomic data, simplifying the process of analyzing and interpreting DNA sequences.
